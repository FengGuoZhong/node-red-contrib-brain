# Install

```bash

$ cd $HOME/.node-red
$ npm install node-red-contrib-brain

```

Or globally alongside Node-RED:

```bash

sudo npm install -g node-red-contrib-brain

```

**You will need to restart Node-RED for it to pick-up the new nodes.**
